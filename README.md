# STAR-MAE
Official implement of paper "Masked Autoencoders for Spatio-Temporal Audio Representations: Theory and Optimization".

Under review. For pre-training data (AudioSet), please refer to our other repository: https://github.com/Jiayu-Xiong/AudioRWKV

All the experiments were conducted on a single RTX4090. In future revisions, we will automatically enable Flash Attention v2. Please ensure that your GPU(s) is of the Ampere architecture or an updated model.

code reference: 
MAE: https://github.com/facebookresearch/mae
AST: https://github.com/YuanGongND/ast
